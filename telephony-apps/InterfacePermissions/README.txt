Apk to be included in baseline image
-------------------------------------
This apk needs to be included as is and should not be re-signed with a different certificate.
The apk is already signed with a certificate that is owned by Qualcomm Technologies Inc.

InterfacePermissions.apk grants secure access to enhanced telephony diagnostic interfaces through signature verification and must be included in the device system image. 
The system grants this permission only if the requesting application is signed with the same certificate as used for this apk.
